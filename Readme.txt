Paper:  Genyi Wan, Zhen Ye, Yusheng Xu, Rong Huang, Yingying Zhou, Huan Xie, and Xiaohua Tong. Multi-Modal Remote Sensing Image Matching Based on Weighted Structure Saliency Feature. 

This package of files is the executable code for the WSSF algorithm. 
The code is being collated and will be given in a month.
The code is compiled by the Matlab platform and the runtime environment requires a version of Matlab equal to or higher than Matlab2019b.
